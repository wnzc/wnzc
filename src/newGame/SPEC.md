# 棋盘游戏合集 - 项目规范

## 项目结构

```
/board-games/
├── index.html        # 主页 - 游戏列表入口
├── tictactoe.html    # 井字棋
├── gomoku.html       # 五子棋
├── othello.html      # 翻转棋
├── connectfour.html  # 四子棋
└── SPEC.md           # 本规范文档
```

## 功能概述

### 主页 (index.html)
- 显示游戏列表：井字棋、五子棋、翻转棋、四子棋
- 点击跳转到对应游戏页面
- 统一的深色渐变背景风格

### 游戏通用功能
- 人机对战模式（AI对手）
- 重新开始按钮
- 胜负/平局弹窗提示
- 响应式设计适配移动端
- 返回主页链接

## 各游戏详情

### 1. 井字棋 (tictactoe.html)
- 3x3 棋盘
- 玩家执 X（红），AI 执 O（青）
- AI 策略：优先赢棋 > 防守 > 中心 > 角落 > 边缘
- 新棋子发光效果

### 2. 五子棋 (gomoku.html)
- 15x15 棋盘
- 玩家执黑棋，AI 执白棋
- 先连成五子获胜
- AI 评分系统：进攻/防守/位置评估
- 手机端适配（小格子显示）

### 3. 翻转棋 (othello.html)
- 8x8 棋盘，绿色棋盘
- 玩家执黑棋，AI 执白棋
- 显示双方棋子数量
- AI 策略：角落优先 > 边缘 > 危险位置规避
- 翻转动画效果

### 4. 四子棋 (connectfour.html)
- 7列 x 6行 竖向棋盘
- 玩家执红棋，AI 执黄棋
- 先连成四子获胜
- 棋子从顶部落下动画
- AI 策略：进攻/防守/中间优先

## 技术实现

- 纯 HTML5 + CSS3 + JavaScript（无框架）
- Google Fonts (Noto Sans SC)
- CSS 动画：落子、翻转、获胜高亮
- 响应式布局适配移动端
- 统一的视觉风格：深色渐变背景、圆角设计

## 视觉风格

- 背景：深蓝紫色渐变 (#1a1a2e → #16213e → #0f0f23)
- 主要色：#FFD93D（黄）、#FF6B6B（红）、#4ECDC4（青）
- 按钮：渐变色 + 悬停动效
- 弹窗：毛玻璃效果 + 边框高亮
